Transfigured Skill Gem Chaos Value Calculator

Use this tool to determine the best skill gem color to transfigure in the lab. It retrieves data from PoeNinja, calculates the value by summing up the prices, and divides by each possible transfigured gem you can obtain. You can also ignore gems below a certain value, assuming they can't be sold, and calculate the value of a gem based on its level and quality. View the source code on GitHub.
